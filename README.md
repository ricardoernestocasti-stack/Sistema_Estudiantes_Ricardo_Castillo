# Sistema_Estudiantes_Ricardo_Castillo
Programa en Python para gestionar estudiantes. Permite agregar, eliminar, actualizar, buscar y mostrar información, además de calcular el promedio general del grupo.

# Uso
Ejecutar en Google Colab o con:
python estudiantes.py

# Funciones
- agregar_estudiante  
- eliminar_estudiante  
- actualizar_estudiante  
- buscar_estudiante  
- mostrar_todos  
- calcular_promedio_general
